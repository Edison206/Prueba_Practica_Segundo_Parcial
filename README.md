# Aplicativo Interactivo de Lógica y Algoritmos
Aplicativo de consola desarrollado en C++ y Java como parte de la prueba práctica de la asignatura Algoritmos y Lógica de Programación — Universidad Técnica de Ambato, FISEI, Carrera de Software. Período enero–julio 2026.

Descripción
El programa presenta un menú interactivo con cuatro opciones:

Operaciones básicas — suma, resta, multiplicación y división con validación de divisor cero.
Registro de notas — ingreso de 5 notas en un arreglo; calcula promedio, nota mayor, nota menor, cantidad de aprobados y reprobados.
Guardar resultados — exporta los resultados al archivo resultados.txt con el nombre del estudiante, fecha y lenguaje utilizado.
Salir — finaliza la ejecución del programa.

Tecnologías

C++17 — compilado con g++ (MinGW)
